# 感应 / 碰撞 / 攻击 — 数值说明

全游戏**只用一个间隙**：`config.interaction.gap`（当前 **44** 像素，表示两物体**表面之间**的空隙）。

## 公式

```
圆心距 ≤ r1 + r2 + gap   → 进入感应圈
  ├─ 同级且可合成       → 合成（Matter 球）
  └─ 否则               → 球体互挤，或 1v1 对战
绘制外圈半径            = 单位半径 + gap
```

## 等级 1 示例（gap=44）

| 对象 | 半径 | 外圈半径 | 与同级/士兵圆心距上限 |
|------|------|----------|----------------------|
| 士兵球 | 20 | 64 | 84 |
| 哥布林 | 14.0 | 58.0 | 与士兵球 78.0 |

敌人显示半径 = 配置半径 × `lane.enemyRadiusScale`（0.78）。

## 调参

只改 `config.js` → `interaction.gap` 一处。
